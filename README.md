**Movie Box Office and Ratings Analysis Dashboard (Excel)
Project Overview******

This project presents a Movie Box Office and Ratings Analysis Dashboard developed using Microsoft Excel. The dashboard is designed to analyze and visualize movie performance data, including box office collections, audience ratings, and genre-based trends. It provides clear and structured insights through interactive visualizations.

**Objectives**
To analyze box office performance of movies
To evaluate and compare movie ratings
To identify trends across genres and release years
To develop an interactive dashboard for effective data analysis
**Tools and Technologies**
Microsoft Excel
Pivot Tables and Pivot Charts
Slicers and Filters
Data Cleaning and Transformation
**Key Features**
Box Office Analysis showing total and average revenue
Ratings Comparison across movies
Genre-wise performance analysis
Identification of top-performing movies
Interactive filtering based on genre, year, and ratings
Dataset Description

**The dataset used in this project includes the following fields:
**
Movie Title
Genre
Box Office Collection
Ratings
Release Year
Methodology
Data collection
Data cleaning (handling missing values and duplicates)
Data transformation
Creation of pivot tables
Development of charts and visualizations
